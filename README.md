# Login Form (Login-Form-120)

A clean, responsive login form built with HTML, CSS, and JavaScript.

![Login form screenshot](./Screenshot%202024-11-17%20185312.png)

## Overview

This repository contains a simple login form UI made using plain HTML, CSS, and a bit of JavaScript for front-end behavior and validation. It's intended as a small, reusable component or a learning example for building forms and styling them with modern CSS.

## Features

- Responsive layout that adapts to different screen sizes
- Clean, modern styling with focused/hover states
- Basic client-side validation (JavaScript) for demonstration
- No external dependencies — works with just a browser

## Demo / Screenshot

The screenshot above shows the login form as it appears in a desktop browser. To see it live, open `index.html` in your web browser.

## Getting Started

To run the project locally:

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-120.git

2. Open `index.html` in your browser (double-click the file or use a live-server extension).

That's it — the form is a static UI and does not require a backend.

## Project Structure

- `index.html` — Markup for the login form
- `style.css` — Styling and layout
- `Screenshot 2024-11-17 185312.png` — Example screenshot used in this README

## Customize

- Change colors, spacing, and fonts in `style.css` to match your design system.
- Extend JavaScript to connect the form to a backend or authentication flow.
